Benvinguts!

Hem creat aquesta plataforma HTML i CSS amb el fi de compartir els nostres treballs de l'assignatura Enginyeria de Software amb el professorat i la resta de companys.

Els idiomes que utilitzarem seran el català i l'anglès, en funció de l'idioma que s'empri a l'hora de demanar-nos la feina.



"Software is a gas: it always expands to
fit whatever container it is stored in."
—Nathan Myhrvold

A continuació es presenta un petit índex amb la estructura principal dels nostres treballs que s'anirà actualitzant a mesura que avança la assignatura. Per tal de veure el contingut de cada pàgina només cal fer clic amb el ratolí sobre el noms, atès que estan hipervinculats a altres pàgines HTML on es troba el contingut específic de cada seminari. Un cop allà, també es disposa d'un botó per a tornar novament a aquesta pàgina.
