Task 2: Domain Requirements.

• Relationship between client and book. There is a domain requirement. The user has to choose which payment system wants to use to pay for the book.

• Relationship between book and Author. An author can upload works and the domain requirement is that he has to upload the book in PDF format.

• Relationship between Client and Author. There is a domain requirement. The client has to choose a way to donate to the author.

• Relationship between Client and Author. There is a domain requirement. The client has to rate the work either leaving a comment or selecting how many stars he thinks that the book diserves.

• Relationship between Editor and Book. There is a domain requirement. When the editor classifies a book he has to choose one of the collections that the website offers him (from adventure to uncategorized).
